# Бот-календарь встреч

Telegram-бот для планирования встреч команды с защитой от пересечения слотов.

## Возможности

- Регистрация сотрудника по /start
- Добавление встречи: название, участники (выбор кнопками), дата и время
- Запрет двойного бронирования: нельзя занять слот, если участник уже занят
- Просмотр: /day (день), /week (неделя), /my (мои встречи) — виден всем

## Важно

Чтобы тебя можно было пригласить на встречу, сначала напиши боту /start —
Telegram не позволяет добавить пользователя, который не запускал бота.

## Стек

Python 3.11, aiogram 3, SQLite (aiosqlite). Деплой на Railway (polling, Procfile).

## Локальный запуск

```
pip install -r requirements.txt
```

Скопируй .env.example в .env и впиши BOT_TOKEN

```
python bot.py
```

## Команды

- /start — регистрация
- /add — добавить встречу
- /day, /week, /my — просмотр
