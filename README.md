# Бот-календарь встреч

Telegram-бот для планирования встреч команды с защитой от пересечения слотов.

## Возможности

- **Инлайн-меню** — `/start` и `/menu` показывают двухуровневую навигацию (Встречи / Анализ рекламы) с кнопками
- **Календарь и тайм-пикер** — при добавлении встречи дата выбирается по календарю (месяц вперёд/назад), время — кнопками часов и минут
- **Автогенерация видеоссылки** — при создании встречи автоматически генерируется ссылка `https://meet.jit.si/...`; показывается при сохранении и в просмотрах
- **Напоминания за 30 мин** — все участники получают ЛС с названием, временем и ссылкой на видеовстречу
- **Запрет двойного бронирования** — нельзя занять слот, если участник уже занят
- **Просмотр**: `/day` (день), `/week` (неделя), `/my` (мои встречи) — виден всем
- **Раздел «Анализ рекламы»** — загрузка CSV-выгрузки кампаний или демо-режим, расчёт ДРР, флаги 🔴/🟢, рекомендации
- **Валидация** — нельзя создать встречу в прошлом, нельзя нажать «Готово» без участников

## Важно

Чтобы тебя можно было пригласить на встречу, сначала напиши боту `/start` или открой меню — Telegram не позволяет добавить пользователя, который не запускал бота.

## Меню и навигация

Главное меню вызывается командами `/start` и `/menu`. Разделы:

- **📅 Встречи** — добавление встречи (+ кнопки выбора участников), просмотр на день / неделю / мои
- **📊 Анализ рекламы** — загрузка CSV или демо-анализ с расчётом ДРР и флагами

Внутри каждого раздела — кнопки действий и кнопка «⬅️ Назад» для возврата.

## Напоминания

За ~30 минут до начала встречи каждый участник получает личное сообщение-напоминание со временем и ссылкой на видеовстречу. Повторно одно и то же напоминание не приходит.

## Анализ рекламы

Загрузи CSV-выгрузку кампаний — бот посчитает ДРР (доля рекламных расходов), пометит убыточные кампании и даст рекомендации (отключить / снизить ставку / масштабировать).

Поддерживаемые заголовки колонок:

| Английский  | Русский    |
|-------------|------------|
| campaign    | Кампания   |
| spend       | Расход     |
| revenue     | Выручка    |
| buyout      | Выкуп      |
| orders      | Заказы     |

Нет файла? Нажми «🧪 Демо-анализ» — отчёт покажется на встроенном примере.

## Стек

Python 3.11, aiogram 3, SQLite (aiosqlite). Деплой на Railway (polling, Procfile).

## Локальный запуск

```
pip install -r requirements.txt
```

Скопируй `.env.example` в `.env` и впиши `BOT_TOKEN`

```
python bot.py
```

## Команды

- `/start` — регистрация и главное меню
- `/menu` — показать главное меню
- `/add` — добавить встречу
- `/day` — встречи на сегодня
- `/week` — встречи на неделю
- `/my` — мои встречи
